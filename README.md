# Course Module – Discovery Side (Frontend)

## Intern Name
Aditya Pal

## Objective
This module focuses on the course discovery experience for users. It allows users to browse available courses, search for relevant courses, and view them in a clean grid-based layout.

## Features Implemented
- Course listing page UI
- Reusable course card component
- Search bar UI with basic filtering logic
- Filter dropdown UI (category/level based)
- Grid layout for courses
- Click on course card navigates to course detail page (UI level)

## API Integration (Planned / Mocked)
- getCourses(): Fetches list of courses to display on the discovery page

## Tools & Tech
- React (Component-based UI)
- Basic CSS for layout and styling
- GitHub for version control

## Status
UI completed. Logic and API integration can be extended as backend becomes available.
